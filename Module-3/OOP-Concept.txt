OOP - Object Oriented Programming

There are the six concept in OOP
1. Class:
-> Class is a collection of data member and member function.
OR
-> Class is a blue print of the object.

2. Object
-> Object is a instance of the class.

- Classes and objects in programming languages are the basic building blocks of object oriented programming.

3. Encapsulation
-> Wrapping up the data

4. Abstraction
-> Hide the data which was unnecessary for the user.

5. Inheritance
-> Reusability of code

6. Polymorphism
